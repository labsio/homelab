# homelab

Self-hosted services running on Proxmox — FreshRSS, linkding, changedetection.
Infrastructure-as-code with Ansible,
services via Docker Compose,
reverse-proxied by Caddy.

## Status

Active migration from private development repository.
Services are being added iteratively with documentation per service.

## Structure

- `ansible/` — configuration management
- `services/` — Docker Compose definitions per service
- `.github/workflows/` — CI for validation and deployment
